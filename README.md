# shopify-boilerplate
Built from the ground up v2.0 skeleton theme for use on the Shopify platform. Based on front-end-boilerplate https://github.com/jmsedwrdmnkme/front-end-boilerplate.git 

Framework-less vanilla, with gulp task runner using pnpm for dependencies/packages, proving optimised output of JS, CSS and images.
